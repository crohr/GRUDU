Automating the installation

After having configured GRUDU you can go on to the final panel telling you that the installation of GRUDU was successful. An option is proposed for the automatic installation of GRUDU on a park of machines. With this file you will not have to go through all the installation process by hand and no GUI will be shown.

Figure 2.5: GRUDU remote hierarchy creation
To use this file you'll just have to type : 

java -jar GRUDU_Installer.jar file_for_automatic_installation.xml
